const int  loops

Plays a sound effect.

This function loads and plays a sound effect from the specified path.

Parameters
pathThe file path to the sound effect file. It should be a null-terminated string.
volumeThe volume level at which to play the sound effect. The value should be within the acceptable range for the audio system.
loopsThe number of times the sound effect should loop. If loops is -1, the sound effect will loop indefinitely.
Returns
A pointer to the Mix_Chunk structure representing the sound effect.
Note
The function will automatically cache the sound effect for future use.
