Darkens a given color by multiplying its RGB components with a specified rate.

Parameters
col{ColRGBA*} Pointer to the ColRGBA structure representing the color to be darkened.
rate{f32} The rate by which to darken the color. A value between 0.0 and 1.0 will darken the color, where 0.0 results in black and 1.0 leaves the color unchanged.

This function takes a pointer to a ColRGBA structure and modifies its RGB components by multiplying each component (red, green, and blue) by the specified rate. The alpha component remains unchanged. This is useful for achieving darker shades of the original color while preserving its transparency.
