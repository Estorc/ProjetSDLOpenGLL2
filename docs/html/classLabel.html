This function initializes a label object with the specified text and alignment parameters.

Parameters
textThe text to be displayed in the label.
haThe horizontal alignment of the text. Possible values:
  • 0: Left-aligned
  • 1: Center-aligned
  • 2: Right-aligned
vaThe vertical alignment of the text. Possible values:
  • 0: Top-aligned
  • 1: Middle-aligned
  • 2: Bottom-aligned
