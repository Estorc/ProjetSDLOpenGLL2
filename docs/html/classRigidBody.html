Detailed Description

Represents a rigid body in the physics simulation.

A rigid body can move and is affected by forces and collisions.

void RigidBody::apply_impulse ( float *  impulse,

float *  correction 
)
inline

Applies an impulse to a rigid body.

This function applies an impulse to a rigid body, affecting its linear and angular velocity. The impulse is applied at the center of mass of the rigid body, and the torque is applied to affect the angular velocity.

Parameters
impulseA pointer to a float representing the impulse to be applied. This should be a 3-element array representing the x, y, and z components of the impulse.
torqueA pointer to a float representing the torque to be applied. This should be a 3-element array representing the x, y, and z components of the torque.
correctionA pointer to a float representing the correction to be applied. This should be a 3-element array representing the x, y, and z components of the correction.
