\hypertarget{getting_started}{}\label{getting_started}\index{Getting Started@{Getting Started}}
This page provides an introduction to setting up and compiling a project using the custom C-\/based game engine.\hypertarget{getting_started_project_context}{}\doxysubsection{\texorpdfstring{Project Context}{Project Context}}\label{getting_started_project_context}
This game engine was developed as part of a second-\/year Computer Science project at the University of Le Mans. The project, Claude Chappe\+: The Infernal Curse of the Silent Shadows, is a first-\/person horror game set in the Claude Chappe computer science building. The player must navigate the eerie halls, avoiding terrifying professors while collecting hidden grades necessary to pass their degree.

The game is built in pure C, using SDL2 and Open\+GL for rendering, with an architecture inspired by Godot.\hypertarget{getting_started_team}{}\doxysubsection{\texorpdfstring{Team}{Team}}\label{getting_started_team}

Once dependencies are installed, compile the project using the provided Makefile commands\+:

Install required libraries\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{make\ install}

\end{DoxyCode}


Build necessary tools\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{make\ tools}

\end{DoxyCode}


Compile the project\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{make}

\end{DoxyCode}


Clean build files\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{make\ clean}

\end{DoxyCode}


Compile in debug mode\+: 
\begin{DoxyCode}{0}

\end{DoxyCode}
