This file contains the implementation of the \doxylink{classCamera}{Camera} class. 

This source file provides the functionality for managing and manipulating camera objects within the application. It includes functions for setting camera parameters, updating the camera\textquotesingle{}s position and orientation, and handling camera-\/related transformations.

The \doxylink{classCamera}{Camera} class is essential for rendering scenes from different viewpoints and is used extensively in the graphics rendering pipeline.
