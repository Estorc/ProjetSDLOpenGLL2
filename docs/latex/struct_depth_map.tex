\doxysubsection{Detailed Description}
Represents a depth map used in rendering. 

This structure encapsulates the necessary components for a depth map, including a frame buffer object (\doxylink{group___framebuffer_ga9d981d4d85307c4db4cf5dc775179a2d}{FBO}), a texture map, and a uniform buffer object (UBO).
