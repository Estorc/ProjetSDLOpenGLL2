This function reads material properties from the specified .mtl file and allocates memory for an array of \doxylink{structMaterial}{Material} structures. The materials are populated with properties such as color, texture maps, and other settings as defined in the .mtl file format.

The following properties are read\+:
\begin{DoxyItemize}
\item Name of the material
\item Ambient, diffuse, and specular colors
\item Specular exponent
\item Optical density
\item Illumination flag
\item Opacity
\item \doxylink{classTexture}{Texture} maps (diffuse, normal, displacement)
\end{DoxyItemize}

If a material with the same name already exists, it will be skipped.

The function dynamically allocates memory for the materials and resizes the allocation if necessary, doubling the size of the materials array when more space is needed.

It utilizes Open\+GL to generate and bind textures for the material\textquotesingle{}s texture maps using SDL for image loading.

The function returns the number of materials loaded or -\/1 if an error occurs.
