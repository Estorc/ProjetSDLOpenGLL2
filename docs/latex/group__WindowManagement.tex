This function calculates the time elapsed since the last update, updates the window\textquotesingle{}s state accordingly, and calls the draw\+\_\+screen function to render the viewport\textquotesingle{}s content. It also enables relative mouse mode for capturing mouse movements. Finally, it swaps the Open\+GL buffers to display the rendered content on the window.
