This enumeration defines the different types of lights that can be used in the rendering system.

\item DIRECTIONAL\+\_\+\+LIGHT\+: A light that emits light in a specific direction, similar to sunlight.
\item SPOT\+\_\+\+LIGHT\+: A light that emits light in a cone shape, with a defined cutoff angle.
